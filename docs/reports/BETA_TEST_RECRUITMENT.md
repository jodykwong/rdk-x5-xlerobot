# Story 1.3 Beta用户测试招募

## 🎯 测试项目介绍

**项目名称**: Story 1.3 - 基础语音识别 (粤语语音识别)
**技术基础**: 阿里云ASR API + 粤语paraformer-v1模型
**测试目标**: 验证粤语语音识别的用户体验和准确性
**测试时间**: 约2小时/用户
**测试地点**: 用户熟悉的家庭或办公环境
**测试设备**: 用户自己的电脑 (Windows/Mac/Linux)

## 📋 测试流程说明

### Phase 1: 环境准备 (30分钟)
1. **设备检查**: 确认电脑和麦克风工作正常
2. **软件安装**: 安装必要的测试软件 (我们提供详细指导)
3. **环境配置**: 配置音频设备和网络连接
4. **基础培训**: 5分钟的测试流程介绍

### Phase 2: 语音识别测试 (45分钟)
1. **基础功能测试**: 5个核心粤语语音指令测试
2. **场景模拟测试**: 日常使用场景的语音交互
3. **准确性验证**: 测试识别准确率和响应速度
4. **用户体验评分**: 为每次识别体验打分

### Phase 3: 深度访谈 (30分钟)
1. **使用体验反馈**: 分享测试过程中的感受和建议
2. **功能需求讨论**: 探讨期望的功能和改进方向
3. **商业化意见**: 对产品商业化的看法和建议
4. **总结建议**: 对产品整体的评价和期待

## 🔧 技术要求和准备

### 设备要求
- **电脑**: Windows 7+, macOS 10.12+, 或 Ubuntu 18.04+
- **内存**: 最少4GB RAM
- **存储**: 至少1GB可用空间
- **网络**: 稳定的互联网连接 (用于API调用)

### 音频设备

- **环境要求**: 相对安静的测试环境

### 软件要求
- **Python**: 3.8+ (如果未安装，我们提供安装指导)
- **浏览器**: Chrome, Firefox, Safari, Edge (任选其一)
- **其他**: 所有测试软件我们都提供
