## 🏗️ 项目架构概览

```
XLeRobot智能语音机器人架构:

┌─────────────────────────────────────────┐
│             用户交互层                    │
│  麦克风输入 ← → 扬声器输出 ← → 可视界面      │
├─────────────────────────────────────────┤
│             核心处理层                    │
│   ASR模块 ← → LLM模块 ← → TTS模块         │
├─────────────────────────────────────────┤
│             云端服务层                    │
│ 阿里云ASR ← → 通义千问 ← → 阿里云TTS       │
├─────────────────────────────────────────┤
│             系统控制层                    │
│    ROS2框架 ← → 系统监控 ← → 资源管理      │
├─────────────────────────────────────────┤
│             硬件加速层                    │
│   RDK X5 NPU ← → 音频设备 ← → 摄像头      │
└─────────────────────────────────────────┘
```

## 🚀 核心特色

### 1. 粤语智能语音交互
- 专业的粤语语音识别和合成
- 自然流畅的粤语对话体验
- "傻强"粤语唤醒词系统

### 2. 云端服务集成
- 阿里云智能语音ASR/TTS服务
- 通义千问大语言模型API
- 智能容灾和降级机制

### 3. 硬件加速支持
- RDK X5 NPU/BPU硬件加速
- 模型推理性能优化
- 边缘计算能力

### 4. 模块化架构
- 基于ROS2的分布式架构
- 清晰的模块边界和接口
- 易于扩展和维护

## 🛠️ 技术栈

### 核心技术
- **操作系统**: Ubuntu 22.04 LTS
- **机器人框架**: ROS2 Humble
- **编程语言**: Python 3.10.12
- **云端服务**: 阿里云智能语音服务
- **硬件加速**: RDK X5 NPU/BPU

### 主要服务
- **ASR服务**: 阿里云智能语音ASR
- **TTS服务**: 阿里云智能语音TTS
- **LLM服务**: 通义千问API
- **通信协议**: HTTP/HTTPS + WebSocket
