```yaml
  - 修订版本: 小错误修复或内容更新
  - 构建版本: 自动构建生成的文档

版本号格式:
  - 格式: v<major>.<minor>.<patch>
  - 示例: v1.0.0, v1.1.0, v1.1.1
```

### 📊 文档状态管理
```yaml
文档状态定义:
  - Draft: 草稿阶段，内容可能不完整
  - Review: 审核阶段，等待审核确认
  - Approved: 已批准，可以作为正式文档
  - Deprecated: 已废弃，不再维护
  - Archived: 已归档，仅供参考

状态转换流程:
  Draft → Review → Approved → Deprecated → Archived
```
