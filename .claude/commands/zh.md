# 全局中文模式

将 Claude Code 设置为全局中文模式，所有对话和运行结果均以中文显示。

## 功能说明

- 所有对话内容使用中文
- 代码解释和技术说明使用中文
- 错误信息和运行结果以中文显示
- 保持代码本身的原样（英文变量名、函数名等）

## 使用方法

输入 `/zh` 即可激活全局中文模式。

## 配置规则

以下是全局中文对话的规则：

1. **对话语言**：所有回复和交流使用中文
2. **代码解释**：用中文解释代码逻辑和功能
3. **错误处理**：错误信息用中文说明
4. **技术术语**：保持专业术语的准确性，必要时提供中英文对照
5. **代码内容**：代码本身保持原样，不翻译变量名、函数名等
